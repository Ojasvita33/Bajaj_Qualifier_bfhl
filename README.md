# BFHL Java API

Spring Boot REST API for the Chitkara BFHL API round that processes an input array and returns categorized numbers, alphabets, special characters, sum, and alternating-caps concatenation.

POST `/bfhl`: https://bajajqualifierbfhl-production.up.railway.app/bfhl  
GET `/health`: https://bajajqualifierbfhl-production.up.railway.app/health
